iOS开发学习笔记2

0x00 概述

本文基于 使用Swift开发iOS8 App实战 1-5章的内容。包含 iOS 开发的几个技巧与控件的使用。

0x01 小技巧

  1. 将应用内的图片资源设置到 UIImageView 控件上

  2. 点击应用中的空白位置隐藏软键盘

0x02 UITextField 用法

  1. 设置 UITextFieldDelegate

  2. 点击回车键时隐藏键盘

  3. 在 storyboard 中可设置 UITextField 的 Return key，使回车键显示指定的文字

0x03 UISegmentedControl 用法

  1. 获取 UISegmentedControl 被选中的项

0x04 UIDatePicker 用法

  1. 获取当前选择的时间

  2. 比较时间、计算时间间隔

0x05 UISlider 用法

  1. 给 UISlider 注册 value changed 事件

0x06 UISwitch 用法

  1. 获取 UISwitch 的状态

0x07 UIPickerView 用法

  1. 实现 UIPickerViewDataSource 协议

  2. 实现 UIPickerViewDelegate 协议

  3. 设置 dataSource 与 delegate

  4. 获取选中的行

0x08 segue 用法

  1. 在 Storyboard 中创建 segue ，并且设置 identifier 属性，方便代码中判断具体的 segue

  2. 处理跳转事件，共享数据

  3. 在 Storyboard 中设置子 ViewController 的 Exit Action，在父 ViewController 中实现 Action ，获取回传数据

0x09 NavigationController 用法

  1. 快速添加 NavigationController ：选中 ViewController ，依次点击 菜单、Editor、Embed In、Navigation Controller，即可快速添加

  2. 在 Storyboard 中可设置 Navigation Item 的 Title

  3. 在代码中设置 Navigation Item 的 Title

  4. 在 Storyboard 中可向 Navigation Item 中添加 Bar Button Item 按钮

0x0A Social Framework 用法

  1. import Social

  2. 设置与打开

0x0B TabBarController 用法

  1. 快速添加 TabBarController ：选中 ViewController ，依次点击 菜单、Editor、Embed Tab Bar Controller

  2. 在 Storyboard 中可设置 Bar Item 中的 Title 以及 Image

  3. 在 Storyboard 中可设置 Tab Bar Item 中的 Badge 角标

  4. 在代码中设置 Tab Bar Item 的属性
